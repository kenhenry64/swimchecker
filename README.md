# Swim Checker Marketing Website

A modern static marketing site for the Swim Checker iPhone/iPad app, using a blue and teal App Store-style visual system.

## Included

- `index.html` — single-page marketing website
- `styles.css` — responsive blue/teal design system
- `script.js` — mobile navigation and reveal animations
- `manifest.json`, `robots.txt`, `sitemap.xml`, `favicon.ico`
- `assets/icons/app-icon-1024.png` — App Store icon asset
- `assets/screenshots/*-1290x2796.png` — 4 App Store portrait screenshots sized 1290 × 2796 px
- `assets/social/og-swim-checker.png` — Open Graph/social preview image

## Deploying

Upload the entire folder to any static host, such as GitHub Pages, Netlify, Vercel, Cloudflare Pages, AWS S3, or GoDaddy hosting.

After deployment:

1. Replace `https://example.com/` in `sitemap.xml` with the live marketing URL.
2. In App Store Connect, paste the live URL into the Marketing URL field.
3. Keep the App Store link in `index.html` as-is unless the app ID changes.
